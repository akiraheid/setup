# System Prompts

System prompts for LLMs.

## Translating for OpenWebUI

System prompts online may be written for other LLM systems like Claude. The following are known mappings to OpenWebUI.

| Service | Tool | OpenWebUI Tool |
| :-----: | :--: | :------------: |
| Claude  | `TodoWrite` | `create_tasks`
| Claude  | `file_read` | `read_file`
