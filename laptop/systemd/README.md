# PC Systemd Services

Software running as a systemd service on a PC.

See the `README.md` in each service's directory for more information about that
service.

## Port assignments

| Port  | Service                  |
| ===== | ======================== |
| 8080  | OpenWebUI                |
| 8081  | MCP Read-only Filesystem |
| 11434 | Ollama                   |
